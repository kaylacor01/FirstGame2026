This file contains the features of the game: Buried Below. Buried Below is a game about a cave explorer who got lost in a cave, and is trying to escape while encountering obstacles (falling boulders). 

During the first iteration of the game, a character, background, objects, and basic player movement was added.

Within the second iteration of the game, the game over screen was added. A main menu was also added which contains two buttons: Play and Settings. 
The Play button allows for the player to start the game. The settings button is the one made to control player speeds.
